# Trabalho-Front-end-II-

Este projeto demonstra habilidades essenciais de desenvolvimento Frontend, abrangendo:

Estrutura HTML Semântica: Uso de tags apropriadas para acessibilidade e SEO.

Design Responsivo: Layout adaptável a diferentes tamanhos de tela (desktop, tablet e mobile) utilizando Media Queries e/ou Flexbox/Grid.

Validação de Formulário: Implementação de regras de validação para garantir a integridade dos dados antes do envio.

Manipulação de Eventos DOM: Captura do evento de submissão e manipulação do feedback para o usuário.

O formulário implementado possui as seguintes funcionalidades:

Validação em Tempo Real: Feedback visual imediato ao usuário sobre a validade dos dados inseridos (ex: formato de e-mail incorreto, campo obrigatório vazio).

Impedimento de Submissão Inválida: O botão de submissão é bloqueado ou o envio é prevenido caso haja erros nos campos obrigatórios.
